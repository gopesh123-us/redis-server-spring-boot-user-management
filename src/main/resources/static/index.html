Welcome to User Management with Spring Boot and Redis Server!

Redis server is a fast in-memory database used for multiple reasons, like real-time data processing and analysis, caching, and message broker. This application is primarily a set of REST APIs for user management, performing CRUD operations. The back-end database is Redis Server, which is a No SQL Database working along side Java Spring Boot. As for the front-end, the application can be tested using Swagger-UI.
